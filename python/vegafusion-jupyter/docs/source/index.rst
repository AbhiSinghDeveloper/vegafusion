
vegafusion_jupyter
=====================================

Version: |release|

Altair Jupyter Widget library that relies on VegaFusion for serverside calculations


Quickstart
----------

To get started with vegafusion_jupyter, install with pip::

    pip install vegafusion_jupyter

or with conda::

    conda install vegafusion_jupyter
